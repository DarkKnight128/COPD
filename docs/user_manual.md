# COPD MVP 用户使用手册

## 登录

打开 `http://127.0.0.1:8001`，使用本地演示账号登录。

| 角色 | 可做事项 |
| --- | --- |
| 管理员 | 导入数据、删除患者、查看导入日志、查看操作日志、查看演示账号 |
| 医生 | 查看患者、触发评估、查看证据、复核报告、编辑报告、确认/驳回报告、导出报告 |
| 科研人员 | 只读查看患者、时间轴、评估结果和报告状态 |

## 管理员流程

1. 登录 `admin`。
2. 点击“导入”，上传固定模板 Excel。
3. 在导入结果页查看患者数、就诊数、检查数、用药数、随访数等统计。
4. 如果导入失败，查看 sheet、行号、字段和失败原因。
5. 在患者列表中按风险、评估状态、复核状态、报告状态、随访状态、导入批次筛选。
6. 如需清理演示数据，勾选患者后点击“删除选中”。
7. 点击“操作日志”查看登录、导入、删除、评估、报告相关动作。

## 医生流程

1. 登录 `doctor`。
2. 进入患者列表，筛选高风险或待复核患者。
3. 打开患者详情，查看基础信息、肺功能、症状评分、CT 摘要、用药、急性加重和随访。
4. 点击“本地规则评估”或“API 智能评估”。
5. 在智能评估结果页查看数据质量、表型、风险、模型调用、关键证据和安全检查。
6. 点击“提交复核”或“编辑报告”。
7. 修改报告正文，每次保存都会生成新版本。
8. 使用同一个复核意见框填写意见，然后点击“确认报告”或“驳回报告”。
9. 点击“导出报告”，使用浏览器打印为 PDF。

## 科研人员流程

1. 登录 `researcher`。
2. 查看患者列表、患者详情、时间轴和评估结果。
3. 查看风险、表型、关键证据、模型版本和数据版本。
4. 科研人员不能导入、删除、触发评估、编辑报告或复核报告。

## 安全边界

- 报告只作为慢阻肺智能辅助评估草稿。
- 报告不替代医生临床判断。
- 系统不输出具体治疗方案或处方建议。
- CT 信息只来自 CT 报告或已提取影像特征。
- mNGS 仅作为感染相关线索。
